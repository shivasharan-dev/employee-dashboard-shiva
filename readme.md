# Employee Management Dashboard

# Project Overview

This project is a simple Employee Management Dashboard built using HTML, CSS, and JavaScript. It demonstrates basic CRUD operations for managing employee records.

# Features

* Add employee details
* Edit employee information
* Delete employee records
* Search employees
* Local storage used to simulate database persistence

# Technologies Used

* HTML5
* CSS3
* JavaScript
* GitHub
* Netlify

# Deployment

The project is deployed using Netlify.
